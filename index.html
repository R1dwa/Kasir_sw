ℹ️ Soal mode offline: Install sebagai aplikasi (di atas) selalu bisa dilakukan. Tapi mode offline penuh (tetap bisa transaksi tanpa internet) baru aktif kalau app ini di-hosting di web server (GitHub Pages, Netlify, dll) — bukan dibuka langsung dari file HTML di HP.
